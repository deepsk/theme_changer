# Starter app

A simple starter app with a drawer and a dynamic theme changer.

- BLoC pattern
- Dynamic theme changer
- Drawer with tiles and expansion tiles
- Settings page


### Packages used:

- [Frideos-flutter](https://pub.dartlang.org/packages/frideos)
